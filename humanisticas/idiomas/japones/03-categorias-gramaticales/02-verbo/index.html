02 Verbo

Clase 3.2: Los verbos y sus conjugaciones

Conceptos básicos

Antes de explicar algo del japonés, quiero explicarlo en español, porque quizás así entiendas mejor.

En español, hay 3 terminaciones que consideramos “infinitivo”: ar, er, ir.
Son la forma más básica de todo verbo. No implican tiempo, número, persona o intención; solo indican la acción.
“Correr”, “Comer”, “Vivir”. ¿Quién “come”? ¿Cuándo “corre”? ¿Quiénes “viven”? No se sabe, porque la versión infinitiva de un verbo es la básica que encuentras en un diccionario.

Esto me lleva a lo que en japonés se conoce como:

Forma Diccionario

En japonés, existen más de tres terminaciones “infinitivas”. En total, hay 11, que se agrupan en tres tipos de verbos. Esto determina cómo vas a conjugar cada verbo.

Grupo Godan (五段ごだん)

Los verbos del grupo Godan tienen 9 posibles terminaciones: う、つ、る、く、ぐ、ぬ、ぶ、む、す.
Todas las formas de diccionario de estos verbos terminan con la vocal “u”.
La forma de conjugarlos es fácil de recordar: la consonante de la sílaba se conserva, pero la vocal cambia según la conjugación.

Por ejemplo:

  • 行いく → “ir” → 行いきます → “voy / iré”
  • 切きる → “cortar” → 切きります → “corto / cortaré”

Aunque algunos verbos como 切きる pueden parecer del grupo Ichidan, realmente son Godan.

Grupo Ichidan (一段いちだん)

Este grupo tiene solo dos terminaciones: “eru” y “iru”.
Si un verbo termina en estas sílabas, es un verbo Ichidan.

Ejemplo:

  • 食たべる → “comer” → 食たべます → “como / comeré”

Conjugar un verbo Ichidan es simple: solo sustituyes “る” por la terminación correspondiente.

Verbos Irregulares

Hay solo dos verbos irregulares:

  • する → “hacer”
  • 来くる → “venir”

Sus conjugaciones son únicas y deben memorizarse por separado.

Lista de Conjugaciones

La lista de conjugaciones, resumida, es:

  1. Forma Diccionario → presente/futuro
  2. Forma Formal → (I) + ます
  3. Forma Negativa → (A) + ない
  4. Forma Te → (U) + って
  5. Forma Pasada → (U) + った
  6. Forma Potencial → (E) + る
  7. Forma Condicional → (E) + ば
  8. Forma Pasado Condicional → (Pasado) + ら
  9. Forma Volitiva → (O) + う
  10. Forma Pasiva → (A) + れる
  11. Forma Causativa → (A) + せる
  12. Forma Pasiva Causativa → (A) + せられる
  13. Forma Prohibitiva → (U) + な
  14. Forma Imperativa → (E)
  15. Forma Progresiva → (Forma Te) + いる

Las conjugaciones explicadas

Forma Diccionario: 辞じ書しょ形けい

Es la forma base del verbo. Se usa para presente y futuro.
Todos los verbos en esta forma terminan en una sílaba con vocal “u”.

Aunque 形かた se lee かたち cuando está solo, en compuestos como 辞じ書しょ形けい se lee けい.

El japonés no marca sujeto ni tiempo en el verbo, así que se asume del contexto.

Forma Formal: ます形けい → (I) + ます

Es la forma educada, usada con desconocidos, superiores o en situaciones formales.

Ejemplos:

  • 脅おどす → “amenazar” → 脅おどします → “amenazo / amenazaré”
  • 開あける → “abrir” → 開あけます → “abro / abriré”
  • 助じょ言げんする → “aconsejar” → 助じょ言げんします → “aconsejo / aconsejaré”

Forma Negativa: 否ひ定てい形けい → (A) + ない

Niega la acción.

Ejemplos:

  • 脅おどす → “amenazar” → 脅おどさない → “no amenazo / no amenazaré”
  • 開あける → “abrir” → 開あけない → “no abro / no abriré”
  • 助じょ言げんする → “aconsejar” → 助じょ言げんしない → “no aconsejo / no aconsejaré”

Forma formal negativa: (E) + ません

  • 脅おどします → 脅おどしません → “no amenazo (formal)”

Forma Te: て形けい → (U) + って

Tiene múltiples usos:

  • Pedir (con cortesía):
    行いってください → “Por favorvaya”

  • Gerundio / Acción en progreso:
    行いっている → “está yendo”

  • Acciones consecutivas:
    話はなして聞きく → “hablaryescuchar”

Marqué las palabras en español en esta zona para que se note qué es y qué no es parte del verbo. Ejemplo “Por favor, vaya” es sólo un verbo en japonés, pero “hablar y escuchar” son dos, al igual que en español. Sé que se nota pero hay gente que no lo nota tan fácil.

Forma Pasada: た形けい → (U) + った

Indica acción completada.

Ejemplos:

  • 脅おどした → “amenazó”
  • 開あけた → “abrió”
  • 助じょ言げんした → “aconsejó”

Forma formal pasada: (I) + ました

  • 脅おどしました → “amenazó (formal)”
  • 開あけました → “abrió (formal)”

Forma Potencial: 可か能のう形けい → (E) + る

Indica capacidad. El objeto va con la partícula が, no con を.

Ejemplos:

  • 日に本ほん語ごが読よめる → “Puedo leer japonés”
  • コーヒーこおひいが買かえる？ → “¿Puedes comprar café?”

Para pedir permiso, no uses la forma potencial. Usa:

  • このりんごを食たべてもいいですか
    → ¿Puedo comer esta manzana?

Forma Condicional: 条じょう件けん形けい → (E) + ば

“Si… entonces…”

Ejemplo:
雨あが降ふれば、家いえにいきます。
→ Si llueve, voy a casa.

También se usa en expresiones de obligación:
勉べん強きょうしなければなりません → Debo estudiar.
O, de forma literal: Si no estudio no será(o 'no se volverá')

Forma Pasado Condicional: たら形けい

  • 日に本ほんに行いったら、何なにが問もん題だいになりますのでしょうか？
    → Si voy a Japón, ¿Qué problema habrá?

意い向こう形けい → (O) + う

Expresa intención o propuesta (¡Vamos a…!).

  • 頑がん張ばろう！ → ¡Vamos a esforzarnos!
  • 行いこう！ → ¡Vamos!
  • 食たべよう！ → ¡Vamos a comer!

Forma Pasiva: 受うけ身み形けい → (A) + れる

Cambia las partículas: を → に, は → が

Ejemplo:

  • Activo:
    コロンはアメリカを発はっ見けんした
    → Colón descubrió América
  • Pasivo:
    アメリカがコロンに発はっ見けんされました
    → América fue descubierta por Colón

Forma Causativa: 使し役えき形けい → (A) + せる

Hacer que alguien haga algo

Ejemplo:
勉べん強きょうさせます
→ Haré que estudies

Forma Pasiva Causativa: 受うけ身み使し役えき形けい → (A) + せられる

Fui obligado a…

Ejemplo:
僕ぼくは友とも達だちに手て紙がみを書かかせられました
→ Fui obligado a escribir una carta por mi amigo

Forma Prohibitiva: 禁きん止し形けい → (U) + な

Prohíbe una acción (imperativo negativo fuerte).

Ejemplo:
試し験けん中ちゅうに携けい帯たい電でん話わを出だすな
→ ¡No saques el móvil en el examen!

Forma Imperativa: 命めい令れい形けい → (E)

Órdenes directas (informales y fuertes).

  • 食たべろ → “¡Come!”
  • 寝ねろ → “¡Duerme!”

⚠️ Evita usar esta forma con personas que no conoces bien. Es muy brusca.

Forma Progresiva: 進しん行こう形けい → (Forma Te) + いる

Acción en curso.

Ejemplo:
食たべている → está comiendo

Extras

Sustantivo abstracto: “こと” se utiliza para convertir verbos en sustantivos abstractos que representan acciones, eventos o situaciones.

  • Ejemplo: 食たべること → “el acto de comer” o simplemente “comer”

También se puede sustituir por “の”, pero es más informal.

A la acción de usar un verbo como un sustantivo se llama “nominalizar”.

Al nominalizar:

  • Los verbos godan usan la raíz en columna i + の
  • Los verbos ichidan eliminan る y añaden の

Ejemplos:

Verbo (infinitivo) Traducción Forma nominalizada Traducción abstracta
食たべる comer 食たべの la acción de comer
見みる ver 見みの la acción de ver
聞きく escuchar 聞ききの la acción de escuchar
行いく ir 行いきの la acción de ir
来くる venir 来きの la acción de venir
する hacer しの la acción de hacer
書かく escribir 書かきの la acción de escribir
読よむ leer 読よみの la acción de leer
話はなす hablar 話はなしの la acción de hablar
寝ねる dormir 寝ねの la acción de dormir

Recuerda: una sola palabra puede pertenecer a varias categorías gramaticales.
Ejemplo: “rápido”

  • Adjetivo: “Él es rápido”
  • Adverbio: “Él corre rápido”
